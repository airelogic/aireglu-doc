    Changelog

    Welcome to the Aireglu changelog.

    August 19, 2021

    Version 1.26.0 Stream endpoints, SMS receiver

    • Add stream endpoint support
    • Add SMS receiver for gov.notify
    • Additional AireAudit integration
    • Changes to MESH receivers and messaging to satisfy MESH accreditation
    • Fix bug around transaction id's
    • Fix bug around mapping paths with special characters
    • Fix bug around job runner ignoring http verbs
    • Fix bugs around dashboard filtering
    • Fix UI bug around request tasks with no body
    • Fix bug around multiple response tasks and filters
    June 29, 2021

    Version 1.25.0 .net 5, Job start and end dates

    • Upgrade to use .Net 5
    • Allow specifying job start and end date
    • Start adding AireAudit support
    • Remove http editor from redirect task UI
    June 16, 2021

    Version 1.24.6 Bugfix

    • Fix issue around function button appearing
    • Further improve large file handling
    June 15, 2021

    Version 1.24.5 Endpoint list improvements, SFTP

    • Add SFTP behaviour
    • Change to stop storing input in validation result for xml
    • Improve large file handling by mesh receivers
    • Fix JSON date parsing bug
    • Improve XSLT handling multiple parameters
    • Remember user's choice of rows per page in endpoint list
    June 01, 2021

    Version 1.24.3 Endpoint list improvements

    • Add endpoint search and ordering
    • Add permissions support to graphql queries
    • Add fax receiver, replacing fax endpoint type
    • Add support for mesh message compression and specifying subject
    • Add support for mesh receiver limiting to specific message id, workflow id, extra features
    May 13, 2021

    Version 1.24.2 AirePolicy permissions

    • Add permissions support via AirePolicy
    • Jobs UI Improvements
    May 05, 2021

    Version 1.24.1 Adapter SDK improvements

    • Improve and enhance adapters SDK
    • Add LUA adapter
    • Add Multicall adapter
    • Extend AireIdentity integration (optional) to support multi-tenant auth
    May 05, 2021

    Version 1.24.1 Adapter SDK improvements

    • Improve and enhance adapters SDK
    • Add LUA adapter
    • Add Multicall adapter
    • Extend AireIdentity integration (optional) to support multi-tenant auth
    Apr 08, 2021

    Version 1.24.0 AireIdentity authentication

    • First integration with AireIdentity for authentication (optional)
    • Improve usage data logging when user can not be retrieved
    Mar 25, 2021

    Version 1.23.1 Bugfix

    • Fix sporadic issue affecting database connection
    Mar 25, 2021

    Version 1.23.0 Adapters

    • Added adapter extensibility point
    Feb 17, 2021

    Version 1.22.1 Bugfix

    • Fix bug around client container SSL
    • Fix issue around mapped in job schedules
    • Fix issue where status code field was wrongly displayed
    Feb 15, 2021

    Version 1.22.0 Response status code

    • Added ability to specify a mappable status code to a response task
    Feb 9, 2021

    Version 1.21.0 Email poller improvements

    • Add concurrency support to e-mail polling
    • Improve e-mail poller memory use
    • Add support for CRLF line breaks
    • Add sorting to secrets UI
    Jan 13, 2021

    Version 1.19.0 Html editor for pdf tasks

    • Add html editor to pdf tasks
    Jan 11, 2021

    Version 1.18.6 Endpoint response improvements

    • Improve endpoint responses to fix issues around filters, content types
    • Improve JWT auth endpoints for ease of use
    Jan 5, 2021

    Version 1.18.5 Raw html e-mails, map status codes

    • Add raw html editor for e-mail tasks
    • Allow use of secrets within variables
    • Allow mapping endpoint instances and status codes into other tasks
    • Pass content type with "retain status code" results
    • Ensure html e-mails always include correct html tags
    • Improve property mapping to better handle duplicate property names
    • Display an error when saving an endpoint fails
    • Retry policy for e-mail tasks to work around AWS issues
    Dec 17, 2020

    Version 1.18.3 Bugfix - endpoint owner name

    • Fix bug where creating an endpoint with the same name as one owned by another user was disallowed
    Dec 17, 2020

    Version 1.18.2 Gov Notify sender id

    • Add support for specifying gov notify sender id
    • Fix bug around response handler json schema
    • Fix race condition around value converters
    Dec 13, 2020

    Version 1.18.1 Bug fixes - endpoint import, liquid arrays

    • Fix bug around case sensitivity for endpoint import
    • Fix bug around liquid array output
    Dec 7, 2020

    Version 1.18.0 Variables

    • Introduce variables, allowing storing and use of non-encrypted values
    Dec 1, 2020

    Version 1.17.5 Jobs pagination

    • Add pagination and search to jobs UI
    • Further enhancemennt to inernal-only and IP authentication
    Nov 30, 2020

    Version 1.17.4 Ip Auth improvements

    • Enhance internal-only authentication to better support load balancing
    Nov 19, 2020

    Version 1.17.3 Bug fix - repository sync

    • Fix where repository sync failed due to logging issues
    Nov 11, 2020

    Version 1.17.2 Bug fix - endpoint version creation

    • Fix bug around creating new endpoint versions
    Nov 6, 2020

    Version 1.17.1 Bug fixes

    • Fix endpoint deletion bug
    • Allow exporting repository endpoints
    • Correct return status codes for pdf, response handler tasks
    Oct 29, 2020

    Version 1.17.0 XSD Union types, data clean-up

    • Support union types for xml sample generation
    • Disable "debug mode" on import
    • Implement scheduled clean-up for usage and transaction data
    Oct 1, 2020

    Version 1.16.5 Display schema errors

    • Display XSD and json schema errors
    • Improve responsiveness of endpoint deletion
    • Add validation for URL fields
    • Stop auto-ticking deployment optio for newer versions
    • Fix defect with extra empty lines in e-mail bodies
    Sept 15, 2020

    Version 1.16.4 Schema and example generation

    • Added controls to manually regenerate schema or example text
    • No longer automatically overwriting schema or example
    • Ensure e-mail client returns 200 on success
    Sept 10, 2020

    Version 1.16.3 Transparent Forward, PDF Generation

    • Introduce new response handler type - "Transparent forwarder" which will pass on entire response of related http request
    • Introduce task type to generate pdf and output as a base64 string
    • Require login to view input and exception data
    • Allow multiple recipients in e-mail tasks
    • More options around endpoint export for encrypted values
    • MongoDB support for hangfire persistence
    • Add confirmation to applying filter to all tasks
