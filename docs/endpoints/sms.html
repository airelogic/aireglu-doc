SMS Task ​

The SMS task can be used to send a text to a user using either a standard text message (using MessageBird), or a Government message (using GOV.UK Notify). An additional Sender ID is required to use the GOV.UK Notify service, and should be provided to you when you register to use that service.

Aside from the Sender ID, the required information is a phone number to send the message to, and the message itself. The phone number should be in international format. For example, if your phone number is 07912 123 456 in the UK, you should enter +447912123456. Wikipedia has a page listing different country codes that are used.

It is recommended to keep the message short and using basic text (not emoji or other uncommon symbols), as some phones may struggle to display long messages and emoji.
